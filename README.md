# Student Performance & Placement Analysis

## 📊 Project Overview

This project is about understanding how students perform in studies. I analyzed their marks, attendance, and placement status using Python.

## 🛠 Tools Used

* Python
* Pandas
* Matplotlib

## 📁 Dataset

* Total Students: 8
* Details include: Name, Gender, Department, Marks, Attendance, and Placement

## 🔍 What I Did

* Calculated total and average marks
* Found the top-performing student
* Compared performance across departments
* Checked placement status of students
* Looked at how attendance affects marks

## 📈 Key Findings

* Students who attend classes regularly usually score better
* Students with higher marks have better chances of getting placed
* Students with low attendance tend to score less
* Performance is slightly different across departments

## ▶️ How to Run

1. Install required libraries:
   pip install pandas matplotlib
2. Run the Python file:
   python analysis.py

## 📁 Output

* Saved the final dataset as a CSV file
* Created charts to understand the data better
